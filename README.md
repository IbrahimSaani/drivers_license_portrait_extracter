# Driver’s License Portrait Extraction System

A simple Streamlit application that extracts the driver's portrait photo from a driver’s license image using MediaPipe face detection.

## Features
- Upload driver’s license image
- Automatic face detection
- Adjustable detection confidence
- Adjustable crop margin
- Download portrait
- Download all faces as ZIP

## Run Locally

```bash
pip install -r requirements.txt
streamlit run src/app.py
